devlog

作品概要

本作はチームでのシステム開発を効率化させ、開発の軌跡を記録するためのものである。

開発者間での自由な発言を促し、新しいアイデアを生み出す『 Log 』。開発進捗状況を把握し、問題の対処法を記録していく『 Project 』。この2つの機能をひとつにさせたものが、本作『 devlog 』(Developer’s Log)である。

開発期間

約1ヶ月

使用言語・技術

技術仕様・機能

  • レスポンシブデザイン
  • Django REST FrameworkによるRestful化
  • Django-allauthによるユーザー認証
  • Ajax(jQuery)ロングポーリングによるリアルタイム情報表示+SPA化
  • ログ・プロジェクトのCRUD、絞り込みと検索、ページネーション

主な用途

  • プロジェクト進捗状況のフォローに
  • 技術的なHowToの備忘録に
  • 開発者間でのアイデア共有に
